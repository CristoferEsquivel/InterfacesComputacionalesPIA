🧠 CONTEXTO DEL PROYECTO

Diseñar un prototipo de software (alta fidelidad) para una tienda de moda online inspirada en Zara México, rediseñando su interfaz para mejorar la usabilidad según principios de Interacción Humano-Computadora.

El rediseño debe corregir problemas detectados en:

Facilidad de aprendizaje

Consistencia

Flexibilidad

Recuperabilidad

Tiempo de respuesta

Robustez

Adecuación a las tareas

Disminución de carga cognitiva

El flujo debe respetar el modelo conceptual de compra (navegación → selección → carrito → pago → confirmación).

🎨 OBJETIVO DEL DISEÑO

Crear una interfaz moderna, minimalista y elegante, pero priorizando:

Claridad funcional sobre estética editorial

Jerarquía visual fuerte

Navegación intuitiva

Filtros visibles

Retroalimentación clara

Indicadores de estado del sistema
