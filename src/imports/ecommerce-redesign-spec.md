PROYECTO: REDISEÑO DE INTERFAZ E-COMMERCE TIPO ZARA

Diseñar un prototipo de software de alta fidelidad para una tienda de moda online inspirada en Zara México. El objetivo es mejorar la usabilidad detectada en el análisis previo (modelo conceptual y reporte de usabilidad), priorizando funcionalidad sobre estética editorial.

El rediseño debe corregir problemas relacionados con:

Facilidad de aprendizaje

Consistencia

Flexibilidad

Recuperabilidad

Tiempo de respuesta

Robustez

Adecuación a tareas

Disminución de carga cognitiva

El flujo debe respetar el proceso de compra completo:
Inicio → Navegación → Selección de categoría → Filtros → Producto → Carrito → Login (si aplica) → Datos de envío → Pago → Confirmación.

DISEÑAR EN DESKTOP (1440px) Y VERSIÓN RESPONSIVE MÓVIL (390px).
